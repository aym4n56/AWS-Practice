Welcome to Ayman's Website

I deployed this site using Amazon S3 Buckets. Here is the process I used:

  • Created an S3 bucket
  • Disabled "Block Public Access" so the site can be public
  • Applied a Bucket Policy to allow public "GetObject" access, again so the public can see.
  • Enabled "Static Website Hosting" in the bucket properties.
  • Uploaded this file, which is a html file, using the AWS CLI.
